## âš¡ Fast Setup

Get up and running on your local machine in three commands.

**1. Clone:**
```bash
git clone [https://github.com/rishabh0282/Portfolite.git](https://github.com/rishabh0282/Portfolite.git)
cd Portfolite
```

**2. Install dependencies:**

```bash
npm install
```
**3. Run the Server:**

```bash
npm run dev
```
